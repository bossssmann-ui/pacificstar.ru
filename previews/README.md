# Скриншоты страниц сайта Pacific Star

Все скриншоты находятся в этой папке: **`/previews/`** в ветке `copilot/refactor-site-description`.

## Как открыть

**Способ 1 — GitHub (браузер):**
1. Откройте репозиторий: https://github.com/bossssmann-ui/pacificstar.ru
2. Переключитесь на ветку `copilot/refactor-site-description`
3. Нажмите на папку `previews/`
4. Кликните на любой файл `.png` — GitHub покажет его прямо в браузере

