🚀 Запуск сайта Pacific Star

Три шага — сайт будет онлайн

🔍 Почему белый экран с логотипом Tilda?

В папке public_html на сервере Timeweb есть старый файл index.html от Tilda. Когда вы заходите на pacificstar.ru — сервер открывает именно его. Наш новый index.html туда не попал, потому что Timeweb при распаковке ZIP не перезаписывает уже существующие файлы.

1
Скачайте файл check.php на компьютер

Нажмите правой кнопкой → «Сохранить ссылку как»

⬇ Скачать check.php
2
Загрузите check.php на Timeweb

Точно так же как загружали hello.html — в папку public_html

1. Войдите на timeweb.com → Панель управления
2. Нажмите «Файловый менеджер»
3. Откройте папку public_html
4. Нажмите «Загрузить файл» (или «Upload»)
5. Выберите check.php из папки Загрузки
3
Откройте check.php в браузере и нажмите кнопку

Скрипт сам перезапишет старый Tilda index.html нашим

🌐 Открыть check.php

На странице нажмите жёлтую кнопку «🔧 Перезаписать index.html и .htaccess»

→ Появится зелёный блок «✅ index.html перезаписан» → нажмите «Открыть pacificstar.ru»

🎉
Сайт работает!
Если сайт всё ещё показывает Tilda — нажмите Ctrl+Shift+N (режим инкогнито) и откройте pacificstar.ru там.
