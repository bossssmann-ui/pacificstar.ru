📦 Установка сайта на Timeweb

Пошаговая инструкция — без технических знаний

✅ Уже сделано:

✅ Файлы сайта загружены на Timeweb
✅ index.html находится в папке
🔧 Осталось: переместить файлы из подпапки в нужное место — 3 шага ниже
1
Загрузите один маленький файл
🔧 Почему пустая страница? Файлы лежат в папке внутри папки. Нужно вытащить их «наружу». Этот файл сделает всё сам — вам только загрузить его.
  1. Скачайте файл-установщик Нажмите кнопку ниже — скачается маленький файл fix.php

    ⬇️ Скачать fix.php
    Файл сохранится в папку «Загрузки» на вашем компьютере. Файл маленький — 7 KB.
  2. Зайдите в Файловый менеджер Timeweb timeweb.com → Панель управления → Файловый менеджер → папка public_html
  3. Загрузите файл fix.php прямо в папку public_html Нажмите «Загрузить файл» (или перетащите) → выберите скачанный fix.php
    Важно: загружаете именно в public_html, не открывая никакие подпапки. fix.php должен лежать рядом с папкой «pacificstar.ru-copilot-...», а не внутри неё.
2
Запустите установщик в браузере
  1. Откройте браузер (Chrome, Safari — любой) Введите в адресной строке и нажмите Enter:
    https://pacificstar.ru/fix.php
  2. Дождитесь зелёного экрана «Готово!» Установщик сам переместит все файлы. Занимает 2–3 секунды.
    🎉 Готово! Сайт установлен
    ✅ index.html
    ✅ about.html
    ✅ css/
    ✅ js/
    [Открыть сайт →]
  3. Нажмите «Открыть сайт →» на зелёном экране Должен открыться сайт Pacific Star с синей шапкой.
3
Удалите установщик (важно!)
  1. Вернитесь в Файловый менеджер → public_html Найдите файл fix.php
  2. Удалите fix.php Правая кнопка мыши → «Удалить». Этот файл больше не нужен, и оставлять его небезопасно.
