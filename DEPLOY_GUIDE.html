Pacific Star

Инструкция по запуску сайта на Timeweb

⚠️ Почему белый экран с Tilda?

При распаковке ZIP-архива Timeweb не заменяет старые файлы. Старый index.html от Tilda остался в папке. Сервер открывает его — отсюда белый экран и логотип Tilda.

Файл r.php уже лежит в вашей папке (загрузился из ZIP, конфликта не было). Он заменит старый Tilda на наш сайт.

🚀 Что нужно сделать — 2 действия
1
Откройте в браузере этот адрес:
pacificstar.ru/r.php
Вы увидите тёмную страницу с двумя зелёными галочками ✅ ✅ и синей кнопкой «Открыть pacificstar.ru».
2
Нажмите синюю кнопку «🌐 Открыть pacificstar.ru».
Сайт откроется — но браузер может показать старую кэшированную версию.
Поэтому сразу нажмите:
Windows: Ctrl + F5   |   Mac: Cmd + Shift + R
Это очистит кэш и покажет новый сайт с синей шапкой Pacific Star.
❓ Если r.php показал белый экран

Значит PHP на сервере отключён или заблокирован.
Тогда — только ручной способ:

1. Timeweb → Файловый менеджер → public_html
2. Найдите файл index.html → правая кнопка → Редактировать
3. Выделите весь текст (Ctrl+A) → удалите
4. Вставьте новый index.html из GitHub (ссылка ниже) → Сохранить
