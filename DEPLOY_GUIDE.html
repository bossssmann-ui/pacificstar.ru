Что видите?

⚪ Белый экран или «404 Not Found» phptest.php не загружен ИЛИ PHP выключен Загрузить новый архив (Шаг 3)

Шаг 3 — Загрузить новый архив (если Шаги 1-2 не сработали)

3.1 — Скачать новый архив (нажмите правой кнопкой → «Сохранить как»):

⬇ Скачать site-pacificstar.zip

3.2 — Timeweb → Файловый менеджер → public_html

  1. Зайдите в timeweb.com → Мои сайты → Файловый менеджер
  2. Откройте папку public_html
  3. Выделите все файлы (галочка в шапке таблицы) → нажмите «Удалить»
  4. Нажмите «Загрузить файл» → выберите site-pacificstar.zip
  5. После загрузки: правая кнопка мыши на ZIP → «Распаковать» или «Извлечь»
❌ Неправильно
public_html/
  📁 site-pacificstar/
    📄 index.html ← здесь
→ белый экран
✅ Правильно
public_html/
  📄 index.html ← прямо здесь
  📁 css/
  📁 js/
→ сайт работает

3.3 — Проверить: открыть pacificstar.ru/test.html
Синий экран с ✅ = файлы на месте → открываем pacificstar.ru

🤖

Шаг 4 — Автоматический деплой (один раз, и навсегда)

Если сделать этот шаг, я буду загружать сайт на Timeweb автоматически каждый раз, когда делаю изменения. Вам не нужно будет ничего делать руками.

  1. Откройте: github.com/bossssmann-ui/pacificstar.ru/settings/secrets/actions
  2. Нажмите «New repository secret»
  3. В поле Name введите: TIMEWEB_PASSWORD
  4. В поле Secret введите ваш пароль от Timeweb
  5. Нажмите «Add secret»

После этого — каждый мой коммит автоматически обновляет сайт. Больше ничего делать не нужно.
