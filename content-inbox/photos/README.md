# Фото для сайта (Фаза 6)

Сюда можно положить исходники **или** прислать ссылку агенту.

## Google Фото — как передать

Прямого подключения Google Фото к агенту нет. Удобные варианты:

1. **Общая ссылка на альбом** — Google Фото → альбом → «Поделиться» → скопировать ссылку и прислать в чат.
2. **Google Drive** — папка с экспортом → «Поделиться» → доступ «все, у кого есть ссылка».
3. **ZIP-архив** — Google Фото → альбом → ⋮ → «Скачать» / [Google Takeout](https://takeout.google.com/) → загрузить в облако (Яндекс.Диск, Drive) и дать ссылку.
4. **Локально** — распаковать в эту папку `content-inbox/photos/` (тяжёлые файлы в git не коммитятся).

## Что указать к фото

- что на снимке (порт, паром, негабарит, склад…)
- маршрут / город
- год
- можно ли в открытый доступ
- есть ли лица клиентов (нужно размытие?)

Отметьте **лучшие 10** для главной и `cases.html`.
