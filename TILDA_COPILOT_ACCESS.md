# Как связать Tilda с Copilot — простая инструкция без технических терминов

---

## Зачем это нужно?

Сейчас я могу писать код и сохранять его в нашу общую папку на GitHub
(думайте о ней как о **«Яндекс.Диске для кода»**).

Чтобы изменения **автоматически появлялись на вашем сайте в Tilda**,
нужно один раз настроить связь между этой папкой и Tilda.
После настройки схема будет такой:

> **Вы:** «Copilot, добавь блок с отзывами»
> → **Я:** пишу и сохраняю
> → **Сайт обновился сам** ✅

Это займёт **15–20 минут**, платить ничего не нужно (Шаги 1 и 2 — бесплатно).

---

## ШАГ 1 — Открыть нашу папку с кодом (2 минуты)

Наша папка с кодом находится по адресу:

👉 **https://github.com/bossssmann-ui/pacificstar.ru**

Если при открытии сайт просит войти — войдите в свой аккаунт GitHub
(тот самый, на котором зарегистрирован `bossssmann-ui`).

---

## ШАГ 2 — Включить «публикацию» папки (5 минут, бесплатно)

Нам нужно сделать так, чтобы файлы из нашей папки были доступны
из интернета — тогда Tilda сможет их подгружать.

Делается это так:

1. Откройте папку: **https://github.com/bossssmann-ui/pacificstar.ru**

2. Найдите вверху надпись **Settings** (Настройки) — она в горизонтальном меню,
   обычно последняя справа. Нажмите на неё.

3. В левом боковом меню найдите пункт **Pages** (Страницы). Нажмите.

4. Появится раздел «Build and deployment». Под надписью **Source** выберите
   из выпадающего списка значение **Deploy from a branch**.

5. Чуть ниже появятся ещё два выпадающих списка:
   - В первом выберите **main**
   - Во втором оставьте **/ (root)**

6. Нажмите кнопку **Save**.

7. Подождите 2–3 минуты. Затем обновите страницу.
   Появится зелёная надпись с адресом вида:
   `Your site is live at https://bossssmann-ui.github.io/pacificstar.ru/`

✅ Готово! Это значит, что наши файлы теперь доступны из интернета.

## ШАГ 3 — Добавить наши файлы в Tilda (10 минут)

Это нужно сделать **один раз**. После этого любые мои изменения
будут автоматически отображаться на сайте.

### 3.1. Открыть настройки сайта в Tilda

1. Войдите в [tilda.cc](https://tilda.cc)
2. На главном экране найдите ваш сайт и нажмите на кнопку **«Настройки сайта»**
   (иконка шестерёнки ⚙️, или кнопка «Настройки» под превью сайта)

### 3.2. Найти раздел «HTML-код»

1. В настройках сайта нажмите вкладку **«Ещё»** (она может называться «More»)
2. Найдите раздел **«HTML-код в шапке и подвале»** (HTML code)
3. Вы увидите два текстовых поля

### 3.3. Вставить код в первое поле

В поле **«Код перед закрывающим тегом HEAD»** (верхнее поле)
скопируйте и вставьте следующие строки:

```
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/css/style.css">
```

### 3.4. Вставить код во второе поле

В поле **«Код перед закрывающим тегом BODY»** (нижнее поле)
скопируйте и вставьте следующие строки:

```
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/main.js" defer></script>
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/i18n.js" defer></script>
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/currency.js" defer></script>
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/calculator.js" defer></script>
```

### 3.5. Сохранить и опубликовать

1. Нажмите **«Сохранить»**
2. Выйдите из настроек
3. Нажмите кнопку **«Опубликовать»** (или «Publish All Pages»)

✅ Готово! Теперь сайт использует наш код.

## ШАГ 4 — Добавить карту маршрутов на нужную страницу

Карту нужно добавлять отдельно на ту страницу, где она должна быть
(например, на главную или на страницу «Услуги»).

1. Откройте редактор нужной страницы в Tilda
2. Нажмите **«+»** (добавить блок) в том месте, где должна стоять карта
3. В строке поиска блоков введите **T123**
4. Выберите блок **«HTML код»** (он выглядит как серый прямоугольник с угловыми скобками `</>`)
5. Два раза кликните по блоку — откроется окно редактирования
6. Удалите всё, что там написано, и вставьте следующее:

```
<div id="routeMapContainer" style="width:100%;height:560px;border-radius:16px;overflow:hidden;background:#071522;"></div>
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/map.js"></script>
```

7. Нажмите **«Сохранить»**
8. Нажмите **«Опубликовать страницу»**

> ⚠️ Внимание: в режиме редактирования карта выглядит как серый прямоугольник.
> Это нормально! На опубликованной странице она отображается правильно.
> Чтобы проверить — нажмите кнопку «Предпросмотр» или откройте опубликованную страницу.

## ШАГ 5 — Добавить меня (Copilot) как помощника в Tilda

Чтобы я мог сам вносить правки в структуру страниц Tilda
(а не только в код), добавьте меня как соавтора:

1. В **Настройках сайта** найдите пункт **«Соавторы»** (Collaborators)
2. Нажмите **«Добавить соавтора»**
3. Введите e-mail, который вам даст куратор (пришлёт отдельно)
4. Выберите роль **«Редактор»** — это безопасно:
   редактор может менять страницы, но не имеет доступа к оплате и паролю
5. Нажмите **«Отправить приглашение»**

## ШАГ 6 — Подключить AmoCRM к формам Tilda (по желанию)

Когда вы пришлёте мне ключи AmoCRM, я сделаю это сам.
Но вы также можете подключить стандартную интеграцию через Tilda:

1. **Настройки сайта** → вкладка **«Аналитика»** → раздел **«CRM»**
2. Нажмите на логотип **AmoCRM**
3. Введите адрес вашего портала AmoCRM
   (обычно выглядит как `ВашеНазвание.amocrm.ru`)
4. Нажмите **«Подключить»** — откроется окно входа в AmoCRM
5. Войдите — и всё. Теперь каждая заявка с сайта будет создавать сделку в AmoCRM
