## Как это всё работает — в двух словах

> Вы пишете мне задачу → я делаю изменения → они **сами** появляются на вашем сайте.
> От вас нужно выполнить настройку **один раз**. Потом — ничего делать не нужно.

# ✅ ШАГ 1 из 2 — Сделать в GitHub (3 минуты)

> GitHub — это место, где хранится код вашего сайта.
> Нужно сделать его доступным из интернета — как нажать «Поделиться» на документе.

**1.** Откройте в браузере эту ссылку:
```
https://github.com/bossssmann-ui/pacificstar.ru/settings/pages
```

**2.** Если попросит войти — введите логин `bossssmann-ui` и пароль от GitHub.

**3.** Вы увидите страницу «Pages». Найдите надпись **«Source»** — под ней кнопка-список.
Нажмите на эту кнопку → выберите **«Deploy from a branch»**.

**4.** Появятся два выпадающих списка:
- В первом выберите **«main»**
- Второй не трогайте — там уже стоит **«/ (root)»**

**5.** Нажмите кнопку **«Save»**.

**6.** Подождите **2 минуты**. Нажмите **F5** (или Ctrl+R) — обновите страницу.

**7.** Появится зелёная плашка с текстом:
> *Your site is live at https://bossssmann-ui.github.io/pacificstar.ru/*

✅ **ШАГ 1 ВЫПОЛНЕН.** Код сайта теперь доступен из интернета.

> 📸 **Не нашли нужную кнопку?** Сделайте скриншот — пришлите мне. Скажу точно куда нажать.

# ✅ ШАГ 2 из 2 — Сделать в Tilda (5 минут)

> Нужно вставить несколько строк текста в настройки Tilda.
> Tilda после этого будет брать наши файлы и показывать их на сайте.
> **Понимать, что написано в этих строках — не нужно.** Просто скопируйте точно как есть.

**1.** Войдите на **tilda.cc** → откройте ваш сайт → нажмите кнопку **«Настройки сайта»**.

**2.** В настройках найдите и нажмите вкладку **«Ещё»** *(она последняя справа)*.

**3.** Нажмите на пункт **«HTML-код»**.

**4.** Откроется страница с **двумя большими белыми полями** для ввода.

---

### 📋 ПЕРВОЕ ПОЛЕ (верхнее, называется «HEAD»):

Нажмите внутри поля, нажмите **Ctrl+A** (выделить всё), затем вставьте (**Ctrl+V**) вот это:

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/css/style.css">
```

---

### 📋 ВТОРОЕ ПОЛЕ (нижнее, называется «BODY»):

Нажмите внутри поля, нажмите **Ctrl+A**, затем вставьте вот это:

```html
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/main.js" defer></script>
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/i18n.js" defer></script>
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/currency.js" defer></script>
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/calculator.js" defer></script>
```

---

**5.** Нажмите кнопку **«Сохранить»**.

**6.** Вернитесь на главную страницу вашего сайта в Tilda.

**7.** Нажмите кнопку **«Опубликовать всё»** *(или «Publish All Pages»)*.

✅ **ШАГ 2 ВЫПОЛНЕН.** На сайте теперь работает:
- 🌐 Переключатель языков (RU / EN / 中文 / 日本語 / 한국어)
- 📞 Кнопки WhatsApp, Telegram, «Перезвоните мне»
- 💱 Калькулятор курсов валют (по данным Центробанка)
- 🧮 Калькулятор стоимости доставки

# ➕ ШАГ 3 — Добавить карту маршрутов (по желанию)

> Карта вставляется на нужную страницу Tilda отдельно.
> Это как добавить видео с YouTube — через специальный блок.

**1.** В Tilda откройте редактор той страницы, куда хотите карту (кнопка **«Редактировать»**).

**2.** Встаньте курсором в то место, где должна быть карта, и нажмите **«+»**.

**3.** В строке поиска наберите: `T123` → нажмите Enter.

**4.** Выберите блок **«HTML код»** *(серый прямоугольник со значком* `</>`*)*.

**5.** Блок появился на странице. **Нажмите на него дважды** — откроется окно с текстом.

**6.** Выделите весь текст в этом окне (**Ctrl+A**) и удалите его.

**7.** Вставьте вот это (**Ctrl+V**):

```html
<div id="routeMapContainer" style="width:100%;height:560px;border-radius:16px;overflow:hidden;background:#071522;"></div>
<script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/map.js"></script>
```

**8.** Нажмите **«Сохранить»** → затем **«Опубликовать страницу»**.

> ❗ В редакторе Tilda карта выглядит как серый квадрат — это нормально.
> Откройте опубликованную страницу в браузере — карта будет работать.

# ➕ ШАГ 4 — Подключить AmoCRM (когда будете готовы)

> После этого каждая заявка с сайта будет автоматически появляться в AmoCRM —
> с именем клиента, телефоном и типом груза.

**1.** Tilda → **«Настройки сайта»** → вкладка **«Аналитика»**.

**2.** Найдите раздел **«CRM»** → нажмите **«AmoCRM»**.

**3.** Введите адрес вашего портала (пример: `pacificstar.amocrm.ru`).

**4.** Нажмите **«Подключить»** → войдите в AmoCRM.

✅ Заявки с форм на всех страницах идут прямо в AmoCRM.

> ⚡ **Для форм в Личном кабинете** — пришлите мне API-ключ AmoCRM.
> Я сам всё настрою. Вам ничего делать не нужно.
