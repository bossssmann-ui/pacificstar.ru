# Как подключить сайт к Tilda — инструкция для начинающих

---

## Что такое Tilda и зачем нам эта инструкция?

Tilda — это конструктор сайтов. Вы создаёте страницы в нём, оформляете их,
и Tilda публикует их в интернете.

Наш код (карта, калькулятор, переключатель языков, виджет связи и т.д.)
хранится отдельно — в облачной папке на GitHub.
Чтобы всё это работало на вашем сайте в Tilda, нужно один раз
«познакомить» Tilda с нашей папкой.

---

## ЧАСТЬ 1 — Добавить помощника в Tilda

Если вы хотите, чтобы я мог сам заходить в Tilda и редактировать страницы,
сделайте следующее:

**Это как дать дубликат ключа от офиса — я смогу работать,
но не смогу сменить замок или получить доступ к кассе.**

1. Зайдите на сайт [tilda.cc](https://tilda.cc) и войдите в свой аккаунт

2. Найдите ваш сайт и нажмите **«Настройки сайта»**
   (шестерёнка ⚙️ или кнопка «Настройки» под картинкой сайта)

3. В левом меню найдите раздел **«Соавторы»**
   (может называться «Collaborators» — если интерфейс на английском)

4. Нажмите **«Добавить соавтора»**

5. Введите e-mail помощника (вам его сообщат отдельно)

6. Выберите роль **«Редактор»**:
   - ✅ Может редактировать страницы
   - ✅ Может публиковать изменения
   - ❌ Не видит данные об оплате
   - ❌ Не знает ваш пароль

7. Нажмите **«Отправить приглашение»**

Помощник получит письмо на почту и примет приглашение.
После этого ваш сайт появится в его аккаунте.

## ЧАСТЬ 2 — Как вставить наш код в Tilda

### Способ А — Вставка кода для ВСЕГО сайта сразу (рекомендуется)

Этот способ позволяет один раз добавить код, и он будет работать
на всех страницах одновременно.

1. Войдите в [tilda.cc](https://tilda.cc)

2. Откройте **«Настройки сайта»** вашего сайта

3. Найдите вкладку **«Ещё»** (или «More», если на английском)

4. Нажмите **«HTML-код»** (или «HTML Code»)

5. Вы увидите два поля для ввода текста.
   В **верхнее поле** вставьте следующее (скопируйте всю строку целиком):

   ```
   <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/css/style.css">
   ```

6. В **нижнее поле** вставьте следующие четыре строки:

   ```
   <script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/main.js" defer></script>
   <script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/i18n.js" defer></script>
   <script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/currency.js" defer></script>
   <script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/calculator.js" defer></script>
   ```

7. Нажмите **«Сохранить»**

8. Выйдите из настроек и нажмите кнопку **«Опубликовать»**

✅ Готово! Теперь на всех страницах работает переключатель языков,
виджет обратного звонка, кнопки мессенджеров и калькулятор.

---

### Способ Б — Добавить карту маршрутов на конкретную страницу

Карта добавляется отдельно на ту страницу, где должна быть.

1. Откройте редактор нужной страницы в Tilda
   (нажмите «Редактировать» рядом с нужной страницей)

2. Встаньте курсором в то место, где хотите разместить карту

3. Нажмите кнопку **«+»** — появится список блоков

4. В строке поиска введите **T123**

5. Выберите блок **«HTML код»**
   (он выглядит как серый прямоугольник с символами `</>`)

6. Два раза кликните на добавленный блок — откроется окно с текстом

7. Удалите всё, что там написано. Вставьте следующее:

   ```
   <div id="routeMapContainer" style="width:100%;height:560px;border-radius:16px;overflow:hidden;background:#071522;"></div>
   <script src="https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/js/map.js"></script>
   ```

8. Нажмите **«Сохранить»**

9. Нажмите **«Опубликовать страницу»**

> 💡 **Почему карта серая в редакторе?**
> Это нормально. Тilda не показывает HTML-блоки в режиме редактирования —
> только серый прямоугольник. Откройте опубликованную страницу в браузере
> и вы увидите настоящую карту.

---

### Способ В — Добавить код только на одну конкретную страницу

Если нужно что-то добавить только на одну страницу, но не на все:

1. Откройте редактор нужной страницы
2. Нажмите шестерёнку ⚙️ вверху страницы — это настройки страницы
3. Перейдите на вкладку **«Дополнительно»** (или «Advanced»)
4. Найдите поле **«HTML для тега HEAD»**
5. Вставьте нужный код туда
6. Нажмите **«Сохранить»**

## ЧАСТЬ 3 — Подключить AmoCRM к формам на сайте

Когда посетитель заполняет форму на сайте — вы хотите, чтобы
его данные автоматически попадали в AmoCRM как новая заявка.

Tilda умеет это делать сама, без программирования:

1. Откройте **«Настройки сайта»**

2. Найдите раздел **«Аналитика»**

3. Внутри найдите вкладку или кнопку **«CRM»**

4. Нажмите на логотип или название **AmoCRM**

5. Введите адрес вашего портала в AmoCRM.
   Он выглядит примерно так: `вашекомпания.amocrm.ru`

6. Нажмите **«Подключить»** — откроется окно входа в AmoCRM

7. Войдите в AmoCRM — и интеграция настроена!

Теперь каждая форма на сайте Tilda будет создавать новую сделку в AmoCRM.

> ⚡ Для форм, которые написаны мной в коде (не стандартные блоки Tilda),
> нужно будет дополнительно передать мне ключи AmoCRM.
> Я сам настрою — вам ничего делать не нужно.

## ЧАСТЬ 4 — Как хранятся наши файлы

Можете думать об этом так:

| Как в жизни | Что в нашем случае |
|------------|-------------------|
| Яндекс.Диск с файлами | GitHub — папка с кодом сайта |
| Ссылка «Поделиться» | GitHub Pages — файлы открыты для интернета |
| Tilda берёт файлы по ссылке | Tilda подгружает наш CSS и JS автоматически |

Ссылка, по которой Tilda обращается к нашим файлам, начинается так:
`https://cdn.jsdelivr.net/gh/bossssmann-ui/pacificstar.ru@main/`

Эта ссылка работает везде в мире быстро — файлы хранятся на серверах
по всей планете и выдаются с ближайшего к посетителю сервера.
