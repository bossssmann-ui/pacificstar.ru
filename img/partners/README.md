# Логотипы партнёров

Разместите файлы логотипов партнёров в эту папку.

## Требования к файлам

| Параметр | Рекомендация |
|----------|-------------|
| Формат | SVG (предпочтительно) или PNG с прозрачным фоном |
| Размер холста | 200 × 80 px (или пропорционально) |
| Вес файла | не более 60 КБ |
| Цвет | монохромный (тёмный или цветной — CSS автоматически обесцвечивает и восстанавливает цвет при наведении) |

## Как подключить логотип

1. Скопируйте файл в эту папку, например `fesco.svg`.
2. В файле `index.html` (и `about.html`) найдите нужный блок `<!-- ── ПАРТНЁР N ── -->`.
3. Раскомментируйте тег `<img>` и замените `partner-N.svg` на имя вашего файла:

```html
<!-- ДО: -->
<!-- <img src="img/partners/partner-1.svg" alt="Название партнёра 1" loading="lazy"> -->
<span class="partner-placeholder">Партнёр 1</span>

<!-- ПОСЛЕ: -->
<img src="img/partners/fesco.svg" alt="FESCO" loading="lazy">
```

4. Сделайте то же самое в блоке **дублирования** (ниже по коду), чтобы анимация прокрутки оставалась бесшовной.

## Добавление нового партнёра

Скопируйте один блок `<div class="partner-card">` (вместе с его копией в секции дублирования) и вставьте новый. Не забудьте сделать то же в `about.html`.
