## 🔐 Единственное действие: добавить пароль в GitHub

> Зачем это нужно: GitHub должен знать пароль, чтобы загружать файлы на сервер.  
> Пароль хранится зашифрованно — никто, кроме автоматики, его не увидит.

### Сделайте 8 кликов:

**1.** Откройте эту ссылку в браузере:  
👉 https://github.com/bossssmann-ui/pacificstar.ru/settings/secrets/actions

**2.** Нажмите зелёную кнопку **`New repository secret`**

**3.** В поле **`Name`** (Название) введите ровно так:
```
TIMEWEB_PASSWORD
```

**4.** В поле **`Secret`** (Значение) введите ваш пароль от Timeweb  
*(тот пароль, который вы используете для входа в панель Timeweb)*

**5.** Нажмите зелёную кнопку **`Add secret`**

**6.** Готово — страница обновится, в списке появится `TIMEWEB_PASSWORD` ✅

## ▶️ Запустить загрузку сайта

После добавления пароля нужно запустить загрузку вручную **один раз**:

**1.** Откройте:  
👉 https://github.com/bossssmann-ui/pacificstar.ru/actions/workflows/tilda-deploy.yml

**2.** Нажмите серую кнопку **`Run workflow`** (справа)

**3.** Ещё раз нажмите зелёную кнопку **`Run workflow`**

**4.** Подождите 1–2 минуты — увидите зелёную галочку ✅

**5.** Откройте **pacificstar.ru** — сайт работает! 🎉
